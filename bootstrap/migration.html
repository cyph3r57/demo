Main changes compared to version 3

  • A new grid system, more flexibile
  • Little change in form code
  • New class attribute in input-group element. It has dropped .input-group-addon and .input-group-btn for two new classes, .input-group-prepend and .input-group-append.
    • @
  • Suppressing of -xs infix in grid options
    • Extra Small < 576px Small Medium Large Extra Large
    Max Container Width .col- .col-sm .col-md .col-lg .col-xl
  • Dropped panels, thumbnails, and wells for a new all-encompassing component, cards.
  • Dropped Glyphicons icon font. Now it's reccomended to use external library like Font Awesome.
  • New margin and padding classes
  • Images classes renamed:
    • .img-responsive to .img-fluid
    • .img-rounded to .rounded
    • .img-circle to .rounded-circle
  • default color renamed in secondary

    • For instance .btn-default become .btn-secondary

  • Badge and panel now is the same thing so label is dropped.
  • Carousel: the item class becomes carousel-item instead of item
  • Carousel control: the item class becomes carousel-control-prev and carousel-control-next
